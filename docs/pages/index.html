CHIP-8

A modern interpreter for the classic virtual machine

What is CHIP-8?

  • The Virtual Machine: Created by Joseph Weisbecker in 1978 to make game development portable across 8-bit systems.
  • Minimalist by Design: Operating with only 35 opcodes, thus quite easy to implement.
  • Iconic Constraints: Powers legendary titles like Pong, Breakout, and Space Invaders on a tiny 64x32 monochrome canvas.

Architecture

  • Memory: 4KB RAM (0x000 - 0xFFF)
  • Registers: 16 8-bit V0-VF
  • Stack: 16-level call stack
  • Display: 64x32 pixels
  • Input: 16-key hexadecimal keyboard

Instruction Set

Control & Jumps

00E0 CLS Clear screen
00EE RET Return from sub
1nnn JP Jump to nnn
2nnn CALL Call sub at nnn
3xkk SE Vx Skip if Vx == kk
4xkk SNE Vx Skip if Vx != kk
5xy0 SE Vx, Vy Skip if Vx == Vy
9xy0 SNE Vx, Vy Skip if Vx != Vy
Bnnn JP V0 Jump nnn + V0

Math & Logic

6xkk LD Vx Set Vx = kk
7xkk ADD Vx Vx = Vx + kk
8xy0 LD Vx, Vy Vx = Vy
8xy1 OR Vx |= Vy
8xy2 AND Vx &= Vy
8xy3 XOR Vx ^= Vy
8xy4 ADD Vx += Vy (VF=C)
8xy5 SUB Vx -= Vy (VF=B)
8xy6 SHR Vx >> 1
8xyE SHL Vx << 1
Cxkk RND Vx = Rand & kk

I/O & Memory

Annn LD I Set I = nnn
Dxyn DRW Draw Sprite
Ex9E SKP Skip if Key Down
ExA1 SKNP Skip if Key Up
Fx07 LD Vx,DT Vx = Delay Timer
Fx0A LD Vx,K Wait for key
Fx1E ADD I,Vx I += Vx
Fx29 LD F,Vx Set I to Font
Fx33 LD B,Vx Store BCD at I
Fx55 LD [I] Reg Dump to RAM
Fx65 LD Vx,[I] Load Reg from RAM

Features

  • Custom assembly syntax
