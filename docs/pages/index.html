CHIP-8 Emulator

A modern interpreter for the classic virtual machine

What is CHIP-8?

  • Created by Joseph Weisbecker in 1978
  • Virtual machine for games on 8-bit computers
  • Simple stack-based architecture
  • 64x32 monochrome display

Architecture

  • Memory: 4KB RAM (0x000 - 0xFFF)
  • Registers: 16 8-bit V0-VF
  • Stack: 16-level call stack
  • Display: 64x32 pixels
  • Input: 16-key hexadecimal keyboard

Instruction Set

// Example: Draw sprite at (x, y) with n bytes
DXYN - Draw sprite at VX, VY with height N

// Example: Jump to address
JP nnn - Set PC to address nnn

// Example: Call subroutine
CALL nnn - Push PC to stack, jump to nnn

Features

  • Custom assembly syntax
  • Audio input support
  • Debugging tools
  • Cross-platform (Go + WebAssembly)
